# StockChartLite

## motivation:
As a research scientist, I wanted to plot some data.
This app retrieves API data of stock and displays the results.

## installation
npm install

## to start
npm start

##
Once stock symbol is typed (e.g. AAPL, AMZN), the graph is displayed
